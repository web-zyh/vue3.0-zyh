## 安装

```js
// 拉取项目代码
git clone https://github.com/web-zyh/vue3.0-zyh.git

cd vue3.0-zyh

// 如果使用别的包管理工具，可以自行安装
npm install

// 运行
npm start

// 打包
npm build
